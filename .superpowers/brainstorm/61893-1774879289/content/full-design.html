What changed vs today

  • Removed: avatar + author name (was Twitter/Threads-style, felt wrong for a personal tool)
  • Removed: mode pill badge above input (redundant with mode bar)
  • Input now floats in a white card with soft shadow — feels tactile
  • Hint chips moved inside the card, color-coded by type (amber idea, blue meeting, green decision, pink feeling, purple insight)
  • Mode bar uses emoji + warm pill background — active mode fills with brand amber
  • Header gets 🧠 emoji for personality — lowercase "new memory" feels casual, not clinical
  • Voice mode gets pulse-ring mic button in a warm card with transcription preview area
  • Link mode integrates clipboard banner inside the card — less intrusive
  • Photo mode uses dashed card with friendly copy about AI description
